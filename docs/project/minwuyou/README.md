# 一建民无忧——医疗生活服务（小程序）
### 项目简介

<br>

民无忧微信小程序主要是为黑龙江省肇东市建一医院的病人服务的微信小程序，主要有三个功能模块：
+ 医院模块：用户绑定建一医院的就诊卡，可以完成线上挂号，预约项目检查身体，比如：血常规，X光片等
+ 商城模块：用户可以在医院商城中选择商品购买，购买时可以选择自取或者快递
+ 服务模块：用户可以选择不同的服务项目，比如水电服务、家务服务、暖气服务等，填好服务信息，等待上门服务即可
